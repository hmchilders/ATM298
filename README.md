Course Resources for the Environmental Data Science Group Study hosted by the Global Environmental Change Lab
Group discussions: Wednesdays, 2:10-3:00pm

### Link to the Participation Spreadsheet: https://docs.google.com/spreadsheets/d/1_I-tBMiCbHVVA1dv17oWiSoLrAG4jrLbsNxh-aBdEWk/edit?usp=sharing

## Course format:

During the first half of the group discussion, you will share your code diary logs in a roundtable discussion. During the second half of the group discussion, we will cover a different topic related to environmental data science:
Week #1:      Course introduction and DataCamp setup and overview
Week #2:      Introduction to version control through GitHub and discussion of FAIR principles in data science
Week #3:      Capstone project discussion
Week #4-5:    Reading, writing and manipulating Earth system and environmental data 
Week #6-7:    Basic data analysis/statistics
Week #8-9:    Data visualization tools
Week #10:     Capstone project presentations
